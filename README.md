# HR Attrition Predictor Dashboard (Power BI)

**Arnav Mehta | Nov 2025**

## Key Insights
- **1470 employees** analyzed from IBM HR dataset
- **237 left** → **16.1% attrition rate**
- **240 high-risk employees** flagged (≤2 yrs tenure + ≤₹50K salary)
- **Sales**: 31% attrition | **R&D**: 14%

## Features
- **One-click drill-through** to individual employee profile
- 6 DAX measures (Attrition Rate %, High Risk Count, etc.)
- Dynamic slicers (Department, JobRole)
- 3 interactive pages

## Tech Stack
- Power BI Desktop
- DAX, Power Query, Star Schema
- IBM HR Analytics Dataset
